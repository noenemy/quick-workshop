# 모듈 1: 통합 채팅

Amazon Quick의 통합 채팅은 AI 어시스턴트와 상호작용하고 조직 데이터에서 답변을 얻을 수 있는 자연어 방식을 제공합니다.

## 학습 목표

이 모듈을 완료하면 다음을 수행할 수 있습니다:

- Amazon Quick 채팅 인터페이스에 접근하고 탐색하기
- AI의 일반 지식을 활용하여 자연어로 질문하기
- 정책 문서를 업로드하여 즉시 AI 기반 분석 수행하기
- 후속 질문에서 채팅이 맥락을 유지하는 방식 이해하기

## 모듈 개요

이 모듈에서는 Quick의 채팅 인터페이스가 경영진에게 다음을 가능하게 하는 방법을 시연합니다:

- **AI 어시스턴트와 상호작용**: 질문을 하고 AI의 내장 지식을 사용하여 답변 받기
- **문서 분석**: 정책 문서, 보고서, 가이드라인을 업로드하여 즉시 분석하기

**예상 소요 시간**: 10분

## 사전 준비 사항

이 모듈을 시작하기 전에 다음을 확인하세요:

- 워크샵 설정을 완료하고 Amazon Quick에 접근할 수 있는 상태
- Riverside시 시민 서비스 환경에 접근 가능한 상태

## 학습 내용 미리보기

이 모듈에서는 Quick의 채팅 인터페이스가 AI 기반 인사이트에 즉각적으로 접근하는 방법을 확인합니다. 자연어로 질문하는 방법과 즉시 분석을 위해 문서를 업로드하는 방법을 배우게 됩니다. 이러한 기능은 기술적 전문성이나 긴 문서 검토 과정의 필요성을 제거합니다.
