# 모듈 2: Spaces와 지식 관리

Amazon Quick의 Spaces를 사용하면 관련 자산(문서, 대시보드, 지식 베이스, 액션, 토픽)을 컬렉션으로 정리하여 조직의 특정 구성원과 검색, 분석, 공유할 수 있습니다. 조직에게 이는 부서장부터 시의회 위원까지 다양한 이해관계자 그룹을 위한 전용 지식 허브를 적절한 접근 수준으로 만드는 것을 의미합니다.

## 학습 목표

이 모듈을 완료하면 다음을 수행할 수 있습니다:

- Spaces가 전략적 의사결정을 위해 문서, 대시보드, 데이터를 정리하는 방법 이해하기
- 새로 생성된 시민 서비스 운영 Space 탐색하기
- 접근 제어가 적절한 정보 공유를 보장하는 방법 인식하기
- 조직에서 Spaces를 적용할 기회 식별하기

## 모듈 개요

이 모듈에서는 Riverside시가 Spaces를 다음과 같이 활용하는 방법을 시연합니다:

- **지식 중앙화**: 서비스 정책, 예산 보고서, 성과 대시보드, 운영 데이터를 하나의 안전한 위치에 통합
- **접근 제어**: 역할과 책임에 따라 정보를 조회, 편집, 공유할 수 있는 사람을 정의
- **협업 활성화**: 부서장, 분석가, 경영진이 동일한 신뢰할 수 있는 정보로 작업할 수 있도록 지원

**예상 소요 시간**: 10분

## 사전 준비 사항

이 모듈을 시작하기 전에 다음을 확인하세요:

- Riverside시 시민 서비스 환경에 접근 가능한 상태

## 학습 내용 미리보기

이 모듈에서는 조직이 다양한 대상을 위해 중요 정보를 정리하는 방법을 보여주는 시민 서비스 운영 Space를 탐색합니다. Spaces가 정보 사일로를 제거하고, 일선 직원부터 시의회까지 이해관계자가 적절한 보안과 거버넌스를 유지하면서 필요한 인사이트에 접근할 수 있도록 보장하는 방법을 확인합니다.

투명성, 책임성, 정보에 입각한 의사결정과 같은 전략적 목표를 Spaces가 어떻게 지원하는지 이해하는 데 집중합니다.
