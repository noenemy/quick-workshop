# 모듈 4: Quick Sight 분석 및 인사이트

Amazon Quick Sight는 원시 시민 서비스 데이터를 실행 가능한 인사이트로 변환하는 강력한 데이터 시각화 및 분석 기능을 제공합니다. 이 모듈에서는 경영진이 기술적 전문성 없이도 간단한 대시보드를 구축하고 자연어로 데이터를 쿼리하는 방법을 시연합니다.

## 학습 목표

이 모듈을 완료하면 다음을 수행할 수 있습니다:

- 자연어 쿼리를 사용하여 시각화 생성하기
- 데이터에 대한 질문을 통해 간단한 대시보드 구축하기
- Quick Sight가 비기술직 경영진에게 데이터 분석을 접근 가능하게 만드는 방법 이해하기
- 전략적 질문에 답하기 위한 실시간 데이터 쿼리

## 모듈 개요

이 모듈에서는 Riverside시가 Quick Sight를 다음과 같이 활용하는 방법을 시연합니다:

- **자연어로 대시보드 구축**: 단순히 질문하는 것만으로 시각화 생성
- **즉각적인 데이터 쿼리**: 기술적 스킬 없이 전략적 질문에 대한 답변 획득
- **데이터 기반 의사결정**: 운영 및 전략 기획을 위한 실시간 인사이트 접근

**예상 소요 시간**: 10분

## 사전 준비 사항

이 모듈을 시작하기 전에 다음을 확인하세요:

- 모듈 1: 통합 채팅과 모듈 2: Spaces와 지식 관리 완료
- 모듈 3: Research Agent에서 리서치 프로젝트 시작 (백그라운드에서 실행됨)
- Riverside시 시민 서비스 환경에 접근 가능한 상태
- 시민 서비스 데이터가 Space에 추가된 상태

> ℹ️ **참고**: 모듈 3의 리서치는 이 모듈을 진행하는 동안 백그라운드에서 실행됩니다. Quick Sight 연습을 완료한 후 결과를 검토하러 돌아갑니다.

## 학습 내용 미리보기

이 모듈에서는 Quick Sight가 데이터 분석의 전통적인 장벽을 어떻게 제거하는지 확인합니다. 대시보드를 구축하는 데 기술적 스킬을 요구하는 대신, 일반 언어로 질문하기만 하면 Quick Sight가 자동으로 적절한 시각화를 생성합니다.

자연어 기능이 경영진에게 데이터 탐색을 어떻게 접근 가능하게 만드는지 경험합니다. SQL 쿼리도, 복잡한 도구도 필요 없이 즉각적인 답변을 제공하는 직관적인 질문만으로 충분합니다. 이 접근 방식은 더 빠른 의사결정, 더 나은 리소스 배분, 이해관계자를 위한 향상된 투명성을 가능하게 합니다.
