# 시작하기

## Amazon Quick 워크샵 시작하기

Amazon Quick 워크샵에 오신 것을 환영합니다. 이 60분짜리 간결한 실습에서는 Quick이 AI 기반 분석과 자연어 인사이트를 통해 시민 서비스 운영을 어떻게 혁신할 수 있는지 보여드립니다.

## 워크샵 개요

- 소요시간 : 120분
- 대상 : Amazon Quick에 관심있는 기업 고객
- 기술수준 : 100-200 레벨


## 학습 내용

이 워크샵에서는 다음 내용을 실습합니다:

- **자연어 쿼리**: 일반 영어로 시민 서비스 데이터에 대해 질문하기
- **지식 관리**: 조직 전체에서 인사이트를 정리하고 공유하기
- **실시간 분석**: 서비스 트렌드와 성과를 보여주는 대시보드 생성하기
- **전략적 인사이트**: AI 기반 분석이 의사결정을 개선하고 혁신을 촉진하는 방법 이해하기
- **자동화**: 수동 작업 없이 주간 보고서를 생성하는 자동화 워크플로우 만들기

## 사전 준비 사항

- 웹 브라우저와 클라우드 개념에 대한 기본적인 이해
- 기술적 또는 코딩 경험 불필요
- 자가 학습의 경우: 관리자 권한이 있는 AWS 계정


## ESEP Gen AI Workshop 참가자

AWS 주최 ESEP Gen AI Workshop 이벤트에 참석하는 경우, 모든 리소스가 사전 구성된 임시 AWS 계정을 제공받습니다.
