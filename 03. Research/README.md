# 모듈 3: Quick Research

Amazon Quick Research는 여러 데이터 소스를 분석하고 상세한 보고서를 생성하여 포괄적인 리서치를 수행할 수 있게 해주는 Amazon Quick의 기능입니다. Quick Research는 인공지능을 활용하여 웹 검색, 업로드된 파일, 연결된 데이터 Space를 포함한 다양한 소스에서 정보를 수집, 분석, 종합하는 것을 돕습니다.

Riverside시 시민 서비스 디렉터로서, 서비스 제공과 시민 만족도를 개선하기 위한 모범 사례를 연구해야 합니다. Quick Research는 외부 소스와 내부 데이터의 포괄적인 분석을 기반으로 정보에 입각한 의사결정을 돕습니다.

## 학습 목표

이 모듈을 완료하면 다음을 수행할 수 있습니다:

- Amazon Quick Research가 무엇이며 어떤 기능을 제공하는지 이해하기
- 명확하고 효과적인 목표로 새로운 리서치 프로젝트 생성하기
- 웹 검색, 파일, Quick 데이터를 포함한 적절한 리서치 자료 선택하기
- 실행 전 리서치 계획 검토 및 확정하기
- 상세한 리서치 보고서 조회 및 분석하기
- 새로운 정보로 기존 리서치 업데이트하고 버전 관리하기
- 다양한 대상과 용도에 맞는 타겟 요약 생성하기
- 리서치 결과를 적절하게 다운로드하고 공유하기

## 모듈 개요

이 모듈은 다음 학습 모듈을 포함합니다:

- **리서치 시작하기**: 리서치 목표 입력, 자료 선택, 리서치 계획 생성 방법 학습
- **리서치 업데이트**: 업데이트 수행, 버전 관리, 리서치 개선 방법 학습
- **리서치 요약**: 경영진 요약, 일반 공유, 맞춤 요약 생성 방법 학습
- **공유 및 관리자 제어 (선택)**: 다운로드 옵션, 공유 기능, 관리 설정 이해

**예상 소요 시간**: 10분
