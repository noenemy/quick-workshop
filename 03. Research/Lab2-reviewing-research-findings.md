# 리서치 결과 검토하기

다른 작업을 마친 후 서비스 제공 모범 사례에 대한 리서치를 확인하러 돌아옵니다.

## 단계 1: 완료된 리서치 접근

1. 메인 내비게이션에서 **Research**로 이동합니다.
2. 리서치 프로젝트 상태를 확인합니다 - **"Investigation complete"**로 표시되어야 합니다.
3. 완료된 리서치 프로젝트를 클릭합니다.

## 단계 2: 리서치 보고서 탐색

보고서는 선택한 모든 소스의 정보를 구조화된 형식으로 종합합니다.

- 왼쪽 패널의 **Topics** 탭을 사용하여 주요 섹션을 탐색합니다.
- 뒷받침하는 증거와 출처 참조가 포함된 구조화된 형식을 확인합니다.
- 페이지 우측 상단의 **Summarize**와 **Share** 버튼을 확인합니다. Share 버튼을 사용하여 팀원에게 리서치를 배포하거나 PDF/Word 문서를 다운로드할 수 있습니다.
- **Reading mode**를 시도하여 사이드바를 숨기고 콘텐츠에 집중합니다.

## 단계 3: 인용 확인

인용은 검증과 심층 조사를 위해 원본 자료에 직접 접근할 수 있게 해줍니다.

1. 보고서 전체에서 번호가 매겨진 인용을 찾습니다.
2. 인용 번호를 클릭하여 출처 세부정보를 확인합니다.
3. 팝업에 출처 기사와 원본 페이지로의 하이퍼링크가 표시되는 것을 확인합니다.

## 단계 4: 문장 분석 이해

문장 분석은 리서치 결론의 근거를 보여주며 분석 과정의 투명성을 제공합니다.

1. **Understand the statement** 아이콘(세 개의 가로선과 플러스)을 찾습니다.
2. 보고서의 아무 문장 옆에 있는 아이콘을 클릭합니다.
3. 다음을 보여주는 설명 창을 검토합니다:
   - 해당 문장이 어떻게 도출되었는지
   - 증거 요약
   - 결론의 분석
