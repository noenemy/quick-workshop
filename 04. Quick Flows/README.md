# 모듈 4: Quick Flows

Amazon Quick Flows는 비즈니스 사용자가 일상적인 작업을 개인 및 팀 생산성을 위한 자동화된 워크플로우로 전환할 수 있게 해주는 Amazon Quick의 기능입니다. 모든 사용자가 기술적 스킬이나 코드 작성 없이도 Amazon Quick 내에서 사용할 수 있는 데이터, 인사이트, 액션을 활용하여 목적에 맞는 워크플로우를 생성, 커스터마이징, 공유할 수 있습니다.

Riverside시 시민 서비스 디렉터로서, 시의회에 보내는 주간 서비스 요청 요약 보고서를 자동화해야 합니다. 이 보고서는 현재 수동으로 작성하는 데 2~3시간이 소요되며, 여러 소스에서 데이터를 가져와 경영진 검토용으로 서식을 맞추는 작업이 필요합니다. Quick Flows는 이 반복적인 작업을 몇 분 만에 실행되는 자동화된 프로세스로 전환하는 데 도움을 줍니다.

## 학습 목표

이 모듈을 완료하면 다음을 수행할 수 있습니다:

- Amazon Quick Flows가 무엇이며 어떤 기능을 제공하는지 이해하기
- 자연어 설명을 사용하여 자동화된 워크플로우 생성하기
- 여러 데이터 소스와 AI 생성 응답 결합하기
- 워크플로우에 조건부 로직과 스케줄링 추가하기
- 팀원과 플로우를 공유하고 조직 라이브러리에 게시하기

## 모듈 개요

이 모듈에서는 Riverside시가 Quick Flows를 다음과 같이 활용하는 방법을 시연합니다:

- **보고 자동화**: 수동 보고서 생성을 자동화된 워크플로우로 전환
- **일관성 보장**: 재사용 가능한 플로우로 부서 간 프로세스 표준화
- **시간 절약**: 수 시간의 수동 작업을 몇 분의 자동화 실행으로 단축
- **협업 활성화**: 조직 효율성을 위해 팀 간 워크플로우 공유

**예상 소요 시간**: 10분

## 학습 내용 미리보기

이 모듈에서는 시의회를 위한 주간 서비스 요청 요약 보고서를 자동화하는 플로우를 생성합니다. Quick Flows가 반복적인 수동 작업을 제거하고 일관되고 고품질의 출력을 보장하는 방법을 확인합니다. 플로우는 데이터를 쿼리하고, 트렌드를 분석하고, 인사이트를 생성하고, 전문적인 보고서를 포맷합니다. 이를 통해 자동화가 수 시간의 작업을 2분의 프로세스로 전환하는 방법을 시연합니다.

운영 효율성, 표준화, 더 높은 가치의 업무를 위한 직원 시간 확보와 같은 전략적 목표를 Quick Flows가 어떻게 지원하는지 이해하는 데 집중합니다.
